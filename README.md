# FullStack UI Library App 📚

A full-stack Library Management web application built with modern web technologies. This project helps users manage books and provides a clean and responsive user interface.

## Features

- Add and manage books
- View available books
- Search functionality
- Responsive UI design
- Dynamic data handling
- Full-stack integration

## Technologies Used

### Frontend
- React.js
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Project Structure

FullStack_UI_LibraryApp

├── frontend
│ └── React frontend

├── backend
│ └── Node/Express backend

## Installation

1. Clone repository

```bash
git clone https://github.com/mukesh4945/FullStack_UI_LibraryApp.git
```

2. Go to project folder

```bash
cd FullStack_UI_LibraryApp
```

3. Install dependencies

```bash
npm install
```

4. Run the project

```bash
npm run dev
```
